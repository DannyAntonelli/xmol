What's XMol?

XMol is a web app that uses a deep learning model to predict if given molecules can penetrate the blood-brain barrier. It was developed as an experiment to study Graph Neural Networks and XAI methods on GNNs. The Neural Network was developed using PyTorch and PyTorch Geometric. The methods used are the ones in the DIG library.
